Ubuntu 20.04 NTP 시간 동기화
파일 백업용 서버를 운용하던 중, 파일 업로드 및 수정 시간이 엉뚱한 시간으로 나오는 것을 발견했고, Timezone 변경 및 NTP 동기화를 통해 정상적으로 시간이 출력되도록 하겠습니다. Timezone 변경 NTP를 통해 시간 대역을 맞추기 전, UTC가 기본으로 잡혀있는 timezone을 KST로 변경합니다. sudo timedatectl set-timezone Asia/Seoul NTP(Network Time Protocol) 설치 $ sudo apt-get install ntp 설정 변경 먼저 서버 설정 파일을 열어 ntp pool server를 변경해줍니다. $ sudo vi /etc/ntp.conf # /etc/ntp.conf, configuration for ntpd; see ntp. 더보기
